# Gestão de Consultas - Sistema de Gerenciamento de Consultas Médicas

Este projeto implementa um sistema de **Gestão de Consultas** para agendamento e visualização de consultas médicas, utilizando o padrão de arquitetura **MVC (Model-View-Controller)** em **PHP**.

## Descrição

Este sistema permite que os usuários possam visualizar todas as consultas agendadas e criar novas consultas. O backend foi desenvolvido em PHP puro com o padrão MVC, sem o uso de frameworks externos, utilizando o PDO para interação com o banco de dados.

## Funcionalidades

- **Listagem de Consultas**: Exibe todas as consultas agendadas.
- **Criação de Consultas**: Permite aos usuários criar novas consultas informando os dados como paciente, médico, data, hora e descrição.
